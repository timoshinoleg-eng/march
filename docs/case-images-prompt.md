# Промпт для генерации иллюстраций кейсов chatbot24.su

## Контекст проекта
**ChatBot24** — инженерное бюро автоматизации заявок. Создаём чат-ботов для бизнеса: клиники, онлайн-школы, e-commerce, B2B. Фокус на результате: конверсия, скорость ответа, ROI.

**Целевая аудитория:** Владельцы бизнеса 30-50 лет, decision-makers, которые хотят автоматизировать заявки и не терять клиентов.

**Стилистика сайта:** Тёмная тема (#161b22 фон), изумрудные акценты (#10b981), минимализм, профессионализм.

---

## Технические требования

| Параметр | Значение |
|----------|----------|
| Формат | 16:9 горизонтальный |
| Размер | 1920×1080 px (или 1200×675 для веб) |
| Стиль | Единый для всех 5 кейсов |
| Текст | Только русский или без текста |
| Цвета | Тёмный фон, изумрудные (#10b981) и синие (#3b82f6) акценты |

---

## Общий стиль (единый для всех кейсов)

**Изометрический 3D** или **неоморфизм** — современный, чистый, технологичный. Без фотореализма. Единая цветовая палитра.

**Референсы:**
- Stripe.com illustrations
- Notion style
- Linear.app aesthetic
- Minimalist tech 3D

**Цветовая схема:**
- Фон: глубокий тёмно-синий / графит (#0f172a, #1e293b)
- Акценты: изумрудный (#10b981), циан (#06b6d4)
- Вторичные: белый (#f8fafc), серый (#94a3b8)
- Градиенты: мягкие, изумрудно-синие

## Кейс 1: Сеть клиник в Москве

**Суть:** Telegram-бот для записи на приём. Интеграция с МИС. 85% меньше пропущенных звонков.

**Ключевые элементы:**
- Смартфон с интерфейсом чата (бubbles сообщений)
- Медицинские символы: крест, стетоскоп, календарь
- Часы (быстрая запись)
- Русские фразы в интерфейсе: "Запись на приём", "Выберите дату", "15 октября"
- Фон: абстрактные линии связи

**Композиция:** Смартфон справа, элементы "летают" вокруг слева. Динамично, но упорядочено.

**Промпт:**
```
Isometric 3D illustration, medical chatbot interface on smartphone, dark navy background #0f172a, emerald accents #10b981, floating UI elements: calendar widget, medical cross, chat bubbles with Russian text "Запись на приём", "15 октября", clean minimalist style, soft gradients, professional healthcare technology aesthetic, no text except interface elements, 16:9 aspect ratio --ar 16:9 --style raw --v 6
```

## Дополнительные инструкции

### Для Midjourney v6:
- Используйте `--style raw` для меньшей "артистичности"
- `--stylize 50-100` для контроля стиля
- Генерируйте 4 варианта, выбирайте лучший
- Для апскейла: U1-U4, затем Upscale (Subtle)

### Для Kimi Slides:
1. Создайте презентацию с 5 слайдами
2. На каждом слайде: промпт выше + "неоморфизм, тёмная тема, изумрудные акценты"
3. Скачайте слайды как PNG (16:9)

### Для других AI (DALL-E, Stable Diffusion):
Добавьте к промпту: `professional illustration, no text except specified, clean background, corporate tech style`
