# ChatBot24 — Лендинг с виджетом

## Описание

Landing page для ChatBot24 с интегрированным чат-виджетом.

## Цветовая схема

| Элемент | Цвет |
|---------|------|
| Фон primary | `#021c1b` |
| Фон secondary | `#042f2e` |
| Акцент | `#14b8a6` |
| Текст | `#ffffff` |
| Текст secondary | `rgba(255,255,255,0.65)` |
| Границы | `rgba(255,255,255,0.1)` |

## Структура сайта (12 секций)

1. **Header** — Навигация с фиксацией
2. **Hero** — "Инженерное бюро автоматизации", цена от 49 000 ₽
3. **Problems** — 4 проблемы с цифрами (60%, 30%, 40%, 0)
4. **HowItWorks** — 4 шага работы системы
5. **Pricing** — 3 тарифа: 49к / 129к / 240к
6. **Calculator** — Интерактивный калькулятор потерь
7. **Advantages** — 4 преимущества
8. **Integrations** — 6 иконок технологий
9. **Cases** — 2 кейса с результатами
10. **Process** — 6 шагов работы
11. **FAQ** — Аккордеон с вопросами
12. **Footer** — Контакты и навигация

## Виджет чата

- Плавающая кнопка в правом нижнем углу
- 6-шаговый бриф:
  1. Сфера бизнеса
  2. Каналы связи
  3. Заявок в день
  4. Задачи бота
  5. Примеры ботов
  6. Бюджет
- AI-ответы на вопросы
- Отправка лида в API

## API Endpoints

- `POST /api/agent` — AI ответы
- `POST /api/lead` — Приём заявок
- `GET /api/health` — Health check

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой

Статические файлы в папке `dist/` готовы для деплоя на Vercel.
