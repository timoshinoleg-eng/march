3. Персональные данные, которые обрабатываются

3.1. Автоматически собираемые данные

При посещении сайта https://www.chatbot24.su автоматически обрабатываются технические данные с использованием сервиса веб-аналитики Яндекс Метрика, включая:

  • IP-адрес;
  • данные файлов cookie;
  • сведения о браузере и устройстве;
  • сведения о действиях пользователя на страницах сайта;
  • время посещения;
  • технические параметры сессии.

Указанные данные носят статистический характер и используются исключительно для анализа посещаемости и улучшения работы сайта.

3.2. Данные, предоставляемые добровольно

При обращении через Telegram пользователь может добровольно предоставить:

  • номер телефона;
  • адрес электронной почты;
  • имя;
  • никнейм в Telegram.

Оператор не запрашивает специальные категории персональных данных и не требует избыточных сведений.

4. Цели обработки персональных данных

Персональные данные обрабатываются исключительно в целях:

  • анализа и улучшения работы сайта;
  • обратной связи по инициативе пользователя;
  • обсуждения возможного сотрудничества.

Оператор не осуществляет обработку персональных данных в целях массовых рассылок и рекламы без отдельного согласия.

5. Правовые основания обработки

Обработка осуществляется на основании:

  • статьи 6 Федерального закона №152-ФЗ;
  • добровольного предоставления данных субъектом;
  • законного интереса Оператора в обеспечении функционирования сайта.

6. Передача персональных данных

Оператор не передаёт персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством Российской Федерации.

Обработка статистических данных осуществляется сервисом Яндекс Метрика в рамках его функционала.

7. Хранение данных

Оператор не формирует и не ведёт систематизированную базу персональных данных пользователей сайта.

Контактные сведения, направленные через Telegram, используются исключительно для ответа и не подлежат долговременному хранению.

Персональные данные обрабатываются не дольше, чем это требуется для достижения целей обработки.

8. Права субъекта персональных данных

Субъект персональных данных вправе:

  • получать сведения об обработке своих данных;
  • требовать уточнения, блокирования или уничтожения данных;
  • отозвать согласие на обработку.
